# SmartWord

## 根目录下包含
    * 设计文档（内含API文档），简要说明了程序设计的思路
    * 使用手册，介绍了各大平台如何编译并运行
    * 演示文稿，第14周展示用ppt

## QtSmartWord文件夹 为Qt制作的GUI版本，支持三个平台(Windows, Linux, MacOS),内含源代码以及编译好的可执行文件
    * source 内含源代码及Qt工程文件
    * runtime 内含 Windows, MacOS, Linux, lib四个目录，前三个对应三大平台可执行文件，lib内含词典文件(共三本，不同词汇量)
    * ReadMe.txt 内含编译平台简要介绍

## SmartWord文件夹 为非GUI版本(终端版本)，支持三个平台(Windows, Linux, MacOS),内含源代码以及编译好的可执行文件
    * source 内含源代码以及三大平台Makefile文件
    * runtime 内含 Windows, MacOS, Linux, lib四个目录，前三个对应三大平台可执行文件，lib内含词典文件(共三本，不同词汇量)
    * ReadMe.txt 内含编译平台简要介绍
